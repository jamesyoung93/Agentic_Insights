# Agentic Insights

Autonomous data-driven discovery system that generates rigorous, evidence-based reports through iterative analysis cycles.

## Quick Start

### Installation

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run streamlit_app_ULTIMATE.py
```

## What's Included

- **streamlit_app_ULTIMATE.py** - Complete interactive Streamlit application
- **auto_enhanced_report.py** - Report generation with statistical analysis
- **world_model_builder.py** - Discovery state and trajectory tracking
- **config.py** - Configuration settings
- **data/** - Sample datasets
